## How to use in Loop
1. Create a Loop workspace called **“ManagerOS – Derm”**.
2. Create pages matching the filenames (or just paste each file into a new Loop page).
3. Keep headings and table column names consistent — it makes Copilot prompts and future automation much easier.

## How to use in OneNote
- Create a notebook called **“ManagerOS Archive”**.
- Paste the pages you want to keep private (e.g., **People notes**, **Journal**).
- Link back to Loop pages for “live” team-facing content.

## Your default daily workflow
- **Teams/Outlook → Copilot extract → Planner tasks → Loop publish → OneNote reflect**
