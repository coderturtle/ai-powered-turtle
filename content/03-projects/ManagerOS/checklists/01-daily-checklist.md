# 01 – Daily Checklist (pin this)

## Start of day (20–25 mins)
- [ ] Outlook: summarize top 5–10 threads → extract actions (owner + due date)
- [ ] Paste actions into **00 – Daily Command**
- [ ] Planner: create tasks in the right buckets
- [ ] Set **Top 3 outcomes**
- [ ] Block a **hands-on** focus window on your calendar

## Before each meeting (2 mins)
- [ ] Define the **decision or outcome** needed
- [ ] If you’re running it, enable transcription when appropriate

## After each meeting (3–5 mins)
- [ ] Copilot: decisions + actions + owners + due dates
- [ ] Log decisions in **90 – Decision Log**
- [ ] Convert actions into Planner tasks
- [ ] Update **99 – Open Loops** if you’re waiting on something

## Midday reset (10 mins)
- [ ] Re-check Top 3 outcomes
- [ ] Defer/kill low-value tasks
- [ ] Protect another hands-on block if needed

## End of day (10–15 mins)
- [ ] Close loops: everything is either a Planner task or dropped
- [ ] Draft/update stakeholder update (if needed)
- [ ] OneNote: 3 bullets → what worked / what didn’t / what to try tomorrow

## Mini prompt cheatsheet
**Meeting recap:** “Decisions, actions (owner+date), unresolved, risks.”  
**Update draft:** “Use pages 30/40/50 + 90 + 99; write for [audience] in [word count].”
