# Real Software & Cohorts

## Real software criteria
- [ ] Feature-flagged
- [ ] Telemetry on
- [ ] Rollback in place
- [ ] User can complete a real task end-to-end

## Cohort selection
- [ ] 5–10% of target population
- [ ] Diversity across advisor types
- [ ] Exclude high-risk accounts
- [ ] Define success & stop rule
